## Project Description
This project is a fully functional To-Do List web application developed using HTML, CSS, and JavaScript.  
The application allows users to add, organize, edit, and manage tasks efficiently with date and time scheduling support.

## Features
- Add new tasks with date and time
- Edit existing tasks
- Mark tasks as completed
- Delete tasks from the list
- Visual distinction between pending and completed tasks
- Real-time task list updates using DOM manipulation
- Input validation and error handling
- Responsive design for all screen sizes
- Smooth UI transitions and hover effects

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation & Event Handling)

## Key Learnings
- Dynamic DOM manipulation
- Event handling for multiple user actions
- Managing task state using JavaScript
- Implementing date and time inputs
- Building responsive and user-friendly interfaces

## How to Run the Project
1. Download or clone the repository.
2. Open the `ToDoApp.html` file in any modern web browser.
3. Add, edit, and manage tasks directly in the browser.
